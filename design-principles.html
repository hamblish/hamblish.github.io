Design Principles

White Space / Clean Design

Google Search

Google's Search website uses a compact, but space filled layout that uses white space to break up each of the search results and divide those results up into blocks of information.

PARC: Repetition

Best Buy

Best Buy's website uses a common color throughout the page, that color being blue, in this specific example they use repetition also in this set of icons. Each icon is in a similar style to the others to provide a consistent theme for the user.

Hick's Law

Wikipedia

Wikipedia greatly reduces the number of immediately visable options available to the user on their mobile website. On their Homepage they show daily content with the header being linited to an expanding menu and a search icon. With the search being a primary reason users visit Wikipedia.
